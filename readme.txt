== Description ==     

List Posts is a plugin show latest posts in widget.

The "List Posts" widget.

### Settings:
1. Category: All, one category.
2. Displays: number posts. 
3. Order (ID, Title, Description, ...) and (meta value)
4. Order by (DESC, ASC)

#### Thank you for your interest in the List Posts plugin.

This section describes how to install the plugin and get it working.

1. Upload folder 'plugin-name' to the `/wp-content/plugins/` directory.
